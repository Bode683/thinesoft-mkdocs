# TinyShop

Welcome to the TinyShop documentation. TinyShop is a lightweight e-commerce solution for small to medium-sized businesses.

## Overview

TinyShop provides everything you need to set up and run an online store with minimal complexity and maximum efficiency.

## Key Features

- **Easy Setup** - Get your store running in minutes
- **Payment Integration** - Multiple payment gateways supported
- **Inventory Management** - Track products and stock levels
- **Mobile Responsive** - Works perfectly on all devices
