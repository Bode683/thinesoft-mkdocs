## Report a Bug

Found a bug? Help us improve by reporting it:

1. Check if the issue is already reported
2. Gather relevant information (version, steps to reproduce, etc.)
3. Submit via email to bugs@thinesoft.com or through our support portal

### Bug Report Template

```
Product: [WiWebb/Mobility/TinyShop]
Version: [Version number]
Description: [Brief description of the issue]
Steps to Reproduce:
1. Step one
2. Step two
3. Step three
Expected Result: [What should happen]
Actual Result: [What actually happens]
Screenshots: [If applicable]
```
